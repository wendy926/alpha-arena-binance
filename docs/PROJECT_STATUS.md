## 技术栈总结

### 前端技术
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI 库**: 现代化组件库
- **状态管理**: React Hooks

### 后端技术
- **运行时**: Node.js 18+
- **框架**: Express.js
- **数据库**: MySQL 5.7
- **ORM**: 原生 SQL 查询

### 部署技术
- **容器化**: Docker + Docker Compose
- **反向代理**: Nginx (可选)
- **进程管理**: Docker 容器管理
